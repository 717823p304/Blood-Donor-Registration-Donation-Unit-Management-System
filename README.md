BLOOD DONOR REGISTRATION AND DONATION UNIT MANAGEMENT SYSTEM

A console-based Java application developed using Core Java, JDBC, and layered architecture to manage blood donor registration and donation unit transactions efficiently.
This system demonstrates structured application development using Bean → DAO → Service → Main architecture, ensuring clean separation of responsibilities and maintainability.

PROJECT OVERVIEW

The Blood Donor Registration & Donation Unit Management System is designed to:
1. Register and manage blood donors
2. Record donation unit transactions
3. Validate donor details
4. Handle fund or unit-related validations using custom exceptions
5. Connect to a relational database using JDBC
6. Maintain transactional integrity
This project emphasizes clean coding practices, exception handling, and database interaction.

TECHNOLOGIES USED

2. JDBC
